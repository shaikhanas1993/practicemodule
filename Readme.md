# practicemodule
<p>
to download dependencies run : <i>go mod download </i>
</p>

<p>
to create modules from scratch : <i>go mod init modulename</i>
</p>
<p>
to fetch dependecies : <i>go fetch dependencies@version/latest</i>
</p>
<p>
to run : <i>go run main.go</i>
</p>
